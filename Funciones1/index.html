¿Qué es una Función?

Definición

Una función es un bloque de código reutilizable que realiza una tarea específica. Las funciones nos permiten organizar nuestro código, evitar repetición y hacer programas más legibles y mantenibles.

Visualización de una Función
def saludar(nombre): """Esta función saluda a una persona""" mensaje = f"¡Hola, {nombre}!" return mensaje # Llamada a la función resultado = saludar("María") print(resultado) # ¡Hola, María!
Ventajas de usar Funciones
  • Reutilización: Escribes el código una vez y lo usas muchas veces
  • Organización: Divide problemas complejos en partes más pequeñas
  • Legibilidad: Hace el código más fácil de entender
  • Mantenimiento: Facilita la corrección de errores y mejoras
  • Testing: Permite probar partes específicas del código

Sintaxis Básica

def nombre_de_la_funcion(parametros): """Documentación opcional de la función""" # Cuerpo de la función return valor_de_retorno # Opcional
Componentes de una Función
  • def: Palabra clave que indica que estamos definiendo una función
  • nombre_de_la_funcion: Identificador único para la función
  • (parametros): Lista de parámetros entre paréntesis
  • :: Dos puntos que marcan el inicio del cuerpo de la función
  • Cuerpo: Código indentado que se ejecuta cuando se llama la función
  • return: Palabra clave opcional para devolver un valor

Parámetros y Argumentos

Diferencia Clave

Parámetros: Variables definidas en la declaración de la función

Argumentos: Valores reales pasados a la función cuando se llama

1. Parámetros Posicionales
def suma(a, b): return a + b resultado = suma(5, 3) # a=5, b=3
2. Valores por Defecto
def saludar(nombre, saludo="Hola"): return f"{saludo}, {nombre}!" print(saludar("María")) # "Hola, María!"
3. Argumentos con Nombre
def presentar(nombre, edad, ciudad): return f"{nombre}, {edad} años, {ciudad}" info = presentar(edad=25, nombre="Carlos", ciudad="Madrid")
4. *args (Variables Posicionales)
def sumar_todos(*numeros): return sum(numeros) print(sumar_todos(1, 2, 3, 4, 5)) # 15
5. **kwargs (Variables con Nombre)
def mostrar_info(**datos): for clave, valor in datos.items(): print(f"{clave}: {valor}") mostrar_info(nombre="Ana", edad=30)
6. Combinación Completa
def funcion_completa(pos1, pos2, default="valor", *args, **kwargs): # Todos los tipos juntos

Llamar a una Función

Sintaxis de Llamada

Para llamar a una función, usa su nombre seguido de paréntesis con los argumentos necesarios.

# Función sin parámetros def decir_hola(): print("¡Hola mundo!") decir_hola() # Llamada a la función # Función con parámetros def multiplicar(x, y): return x * y resultado = multiplicar(4, 7) # Llamada con argumentos
Diferentes Formas de Llamar Funciones
Formas de Llamar la Misma Función
def calcular(a, b, operacion="suma"): # Función de ejemplo # Diferentes formas de llamar: calcular(10, 5) # Posicionales calcular(10, 5, "resta") # Con argumento adicional calcular(a=10, b=5) # Argumentos con nombre calcular(b=5, a=10, operacion="multiplicacion") # Orden diferente

Retorno de Valores

Sin Return (None)
def imprimir_mensaje(texto): print(texto) # No tiene return, devuelve None resultado = imprimir_mensaje("Hola") print(resultado) # None
Return Simple
def calcular_cuadrado(numero): return numero ** 2 cuadrado = calcular_cuadrado(5) # 25
Return Múltiple
def dividir_con_resto(dividendo, divisor): cociente = dividendo // divisor resto = dividendo % divisor return cociente, resto # Tupla cociente, resto = dividir_con_resto(17, 5)
Return Condicional
def evaluar_nota(calificacion): if calificacion >= 90: return "Excelente" elif calificacion >= 80: return "Bueno" else: return "Necesita mejorar"

1. Calculadora Simple

def calculadora(num1, num2, operacion): """ Realiza operaciones matemáticas básicas Args: num1 (float): Primer número num2 (float): Segundo número operacion (str): Tipo de operación (+, -, *, /) Returns: float: Resultado de la operación """ if operacion == '+': return num1 + num2 elif operacion == '-': return num1 - num2 elif operacion == '*': return num1 * num2 elif operacion == '/': if num2 != 0: return num1 / num2 else: return "Error: División por cero" else: return "Operación no válida"

2. Validador de Email

def validar_email(email): """ Valida si un email tiene formato básico correcto Args: email (str): Dirección de email a validar Returns: bool: True si es válido, False si no """ if '@' not in email: return False partes = email.split('@') if len(partes) != 2: return False usuario, dominio = partes if len(usuario) == 0 or len(dominio) == 0: return False if '.' not in dominio: return False return True

Ejercicio: Crear tu Propia Función

Crea una función que convierta temperaturas entre Celsius y Fahrenheit.

def convertir_temperatura(temp, unidad): """ Convierte temperatura entre Celsius y Fahrenheit Args: temp (float): Temperatura a convertir unidad (str): 'celsius' o 'fahrenheit' Returns: dict: Temperatura convertida con detalles """ if unidad.lower() == 'celsius': fahrenheit = (temp * 9/5) + 32 return { 'original': f"{temp}°C", 'convertido': f"{fahrenheit:.1f}°F", 'formula': "°F = (°C × 9/5) + 32" } elif unidad.lower() == 'fahrenheit': celsius = (temp - 32) * 5/9 return { 'original': f"{temp}°F", 'convertido': f"{celsius:.1f}°C", 'formula': "°C = (°F - 32) × 5/9" } else: return {'error': 'Unidad no válida'}
Ejercicio: Función de Contraseña Segura

Crea una función que evalúe la fortaleza de una contraseña.

def evaluar_contraseña(contraseña): """ Evalúa la fortaleza de una contraseña Args: contraseña (str): Contraseña a evaluar Returns: dict: Análisis de la contraseña """ puntuacion = 0 criterios = [] # Longitud mínima if len(contraseña) >= 8: puntuacion += 2 criterios.append("✓ Longitud adecuada (8+ caracteres)") else: criterios.append("✗ Muy corta (menos de 8 caracteres)") # Letras mayúsculas if any(c.isupper() for c in contraseña): puntuacion += 1 criterios.append("✓ Contiene mayúsculas") else: criterios.append("✗ No contiene mayúsculas") # Letras minúsculas if any(c.islower() for c in contraseña): puntuacion += 1 criterios.append("✓ Contiene minúsculas") else: criterios.append("✗ No contiene minúsculas") # Números if any(c.isdigit() for c in contraseña): puntuacion += 1 criterios.append("✓ Contiene números") else: criterios.append("✗ No contiene números") # Caracteres especiales especiales = "!@#$%^&*()_+-=[]{}|;:,.<>?" if any(c in especiales for c in contraseña): puntuacion += 1 criterios.append("✓ Contiene caracteres especiales") else: criterios.append("✗ No contiene caracteres especiales") # Determinar fortaleza if puntuacion >= 5: fortaleza = "Muy Fuerte" color = "verde" elif puntuacion >= 4: fortaleza = "Fuerte" color = "azul" elif puntuacion >= 3: fortaleza = "Moderada" color = "amarillo" else: fortaleza = "Débil" color = "rojo" return { 'puntuacion': puntuacion, 'fortaleza': fortaleza, 'color': color, 'criterios': criterios }

💡 Buenas Prácticas para Funciones

  • Nombres descriptivos: Usa nombres que describan claramente lo que hace la función
  • Una responsabilidad: Cada función debe hacer una sola cosa bien
  • Documentación: Incluye docstrings para explicar qué hace la función
  • Parámetros claros: Usa nombres de parámetros descriptivos
  • Manejo de errores: Considera casos límite y errores posibles
  • Longitud apropiada: Mantén las funciones relativamente cortas y enfocadas
