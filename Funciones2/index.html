1. Parámetros en Funciones

Los parámetros son los valores que una función recibe para trabajar con ellos. Pueden ser obligatorios u opcionales.

1.1 Parámetros Obligatorios

Son aquellos que deben ser proporcionados al llamar a la función. Si no se incluyen, Python lanzará un error.

def sumar(a, b):
    # Esta función suma dos números
    return a + b

¡Pruebalo!

1.2 Parámetros Opcionales

Se definen con un valor por defecto. Si no se proporcionan, se usa el valor predeterminado.

def calcular_carga(peso, factor_seguridad=1.5):
    # Calcula la carga con un factor de seguridad
    return peso * factor_seguridad

¡Pruebalo!

2. Variables en Funciones

2.1 Variables Locales

Solo existen dentro de la función donde se definen.

def calcular_esfuerzo(fuerza, area):
    # 'esfuerzo' es una variable local
    esfuerzo = fuerza / area
    return esfuerzo

2.2 Variables Globales

Se definen fuera de funciones y son accesibles desde cualquier parte del código.

# Variable global
MODULO_ELASTICIDAD = 210000  # MPa para acero

def calcular_deformacion(esfuerzo):
    return esfuerzo / MODULO_ELASTICIDAD

¡Pruebalo!

3. Creación de Módulos

Los módulos permiten organizar código relacionado en archivos separados.

3.1 Estructura de un módulo

Crear un archivo estructuras.py con funciones útiles:

# estructuras.py
def momento_inercia_rectangular(b, h):
    """Calcula el momento de inercia de una sección rectangular"""
    return (b * h**3) / 12

def resistencia_flexion(momento_inercia, distancia_neutro, esfuerzo_admisible):
    """Calcula la resistencia a flexión"""
    return (esfuerzo_admisible * momento_inercia) / distancia_neutro

3.2 Uso del módulo

# main.py
import estructuras

# Calcular para una viga de 200x400 mm
inercia = estructuras.momento_inercia_rectangular(200, 400)
print(f"Momento de inercia: {inercia} mm⁴")

4. Buenas Prácticas en Funciones

4.1 Nombres descriptivos

# Mal
def calc(a, b):
    return a * b

# Bien
def calcular_area_rectangular(ancho, altura):
    return ancho * altura

4.2 Documentación adecuada

def calcular_cortante_maximo(fuerza_cortante, area):
    """
    Calcula el esfuerzo cortante máximo en una sección transversal.
    
    Args:
        fuerza_cortante (float): Fuerza cortante en Newtons
        area (float): Área de la sección transversal en mm²
        
    Returns:
        float: Esfuerzo cortante máximo en MPa
    """
    return fuerza_cortante / area

4.3 Principio de responsabilidad única

Cada función debe hacer una sola cosa y hacerla bien.

Crea una función que calcule la flecha máxima en una viga simplemente apoyada con carga uniforme:

δ = (5 * w * L⁴) / (384 * E * I)

Donde:
w = carga distribuida (kN/m)
L = luz de la viga (m)
E = módulo de elasticidad (MPa)
I = momento de inercia (mm⁴)
